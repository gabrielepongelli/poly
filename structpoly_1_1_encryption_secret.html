Detailed Description

template<std::uint8_t size>
struct poly::EncryptionSecret< size >

An EncryptionSecret is a structure created to hold an initialization vector and a key to use, which size in bytes is specified by the non-type template parameter size. If that parameter is 0 a compile-time error will be raised. This type does not provide any methods useful to handle the data, it's only a structure created to wrap them.
